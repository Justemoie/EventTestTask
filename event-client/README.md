# EventHub - React Client

React + TypeScript клиент для системы управления событиями, интегрированный с ASP.NET Core API.

## Функциональность

### Аутентификация
- Регистрация пользователей
- Вход в систему
- Управление профилем пользователя
- Защищенные маршруты

### Управление событиями
- Просмотр списка всех событий
- Поиск и фильтрация событий
- Создание новых событий
- Редактирование событий
- Удаление событий
- Регистрация на события

### Профиль пользователя
- Просмотр информации о пользователе
- Редактирование профиля
- Статистика активности
- История событий

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **React Router** - маршрутизация
- **Tailwind CSS** - стилизация
- **Lucide React** - иконки
- **Axios** - HTTP клиент
- **Context API** - управление состоянием

## Установка и запуск

### Предварительные требования
- Node.js 16+
- npm или yarn
- Запущенный ASP.NET Core API сервер

### Установка зависимостей
```bash
npm install
```

### Настройка API
Обновите базовый URL API в файле `src/services/api.ts`:
```typescript
baseURL: 'https://localhost:7000/api', // Замените на ваш URL API
```

### Запуск в режиме разработки
```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для продакшена
```bash
npm run build
```

## API Endpoints

Приложение интегрируется со следующими API endpoints:

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `POST /api/auth/logout` - Выход

### События
- `GET /api/events` - Список событий
- `GET /api/events/{id}` - Детали события
- `POST /api/events/create` - Создание события
- `PUT /api/events/update/{id}` - Обновление события
- `DELETE /api/events/delete/{id}` - Удаление события
- `GET /api/events/search` - Поиск событий

### Пользователи
- `GET /api/users/{id}` - Информация о пользователе
- `PUT /api/users/update/{id}` - Обновление пользователя

### Регистрации
- `GET /api/registrations/{eventId}` - Участники события
- `POST /api/registrations/{eventId}/users/{userId}/register` - Регистрация на событие
- `POST /api/registrations/{eventId}/users/{userId}/unregister` - Отмена регистрации

## Разработка

### Добавление новых страниц
1. Создайте компонент в `src/pages/`
2. Добавьте маршрут в `src/App.tsx`
3. Обновите навигацию в `src/components/Header.tsx`

### Добавление новых API методов
1. Добавьте метод в `src/services/api.ts`
2. Обновите типы в `src/types/index.ts`
3. Используйте в компонентах

### Стилизация
- Используйте Tailwind CSS классы
- Добавляйте кастомные стили в `src/index.css`
- Следуйте дизайн-системе
